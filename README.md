## Environment Variables (.env)

| Name               | Description      |
| ------------------ | ---------------- |
| `USERNAME`         | Login UserName   |
| `PASSWORD`         | Login PassWord   |
| `PROJECT`          | Project Slug     |
| `DEFAULT_SWIMLANE` | Default Swimlane |
| `HOST`             | Host URL         |
| `DATABASE_PATH`    | Yaml File Path   |
